static

启动一个纤程

启动纤程不需要特殊创建对象，之需要调用需要在纤程内执行的函数的 start 方法即可：

function func(v1, v2)
{
if(v1 > v2)
console.log(v1);
else
console.log(v2);
}
var fb = func.start(100, 200);
fb.join();
参数
...可变参数序列，此序列会在纤程内传递给函数
返回
返回纤程对象
