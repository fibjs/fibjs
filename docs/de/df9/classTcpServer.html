详细描述

tcp 服务器对象，可方便创建一个标准多纤程 tcp 服务器

使用 TcpServer 对象可以迅速创建一个多纤程并发处理的 tcp 服务器。

function func(conn)
{
var data;
while(data = conn.read())
conn.write(data);
conn.close();
}
new net.TcpServer(8080, func).run();

readonly Stats TcpServer::stats

查询当前服务器运行状态

返回的结果为一个 Stats 对象，初始化计数器如下：

accept : 10, // 上次查询后新建的连接
close : 10 // 上次查询后关闭的连接
}
