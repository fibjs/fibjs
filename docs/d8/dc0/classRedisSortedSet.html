详细描述

Redis 数据库客户端 SortedSet 对象，此对象为包含指定 key 的客户端，只有调用其方法才会操作数据库

用以操作 Redis 的 SortedSet 对象，创建方法：

var db = require("db");
var rdb = new db.openRedis("redis-server");
var set = rdb.getSortedSet("test");
